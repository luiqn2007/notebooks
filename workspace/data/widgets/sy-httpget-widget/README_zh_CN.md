# Http Get 挂件

![preview](./btnView.png)

此挂件可以看作是放在内容里的按钮。可以发起Get请求，配合后端服务对思源进行功能扩展。

**每个挂件所输入的内容都会持久化的。**

**对于删除了的挂件，其持久化的数据，也会在其它挂件使用时进行清理。**
